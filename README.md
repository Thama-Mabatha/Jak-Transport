## Description

### What is it:
This project is a web and mobile-based system called **JAK Move**, developed in 2024 for **JAK Transport**.  
JAK Move is a transport and moving management platform that allows customers to request moving quotes, book jobs, track deliveries in real time, and make payments online.

In addition to customer booking functionality, the system includes:
- A driver mobile application for managing assigned jobs and updating delivery progress  
- A manager dashboard for job oversight, pricing control, payment approval, reporting, and fleet management  

The platform automates manual transport processes, improving efficiency, transparency, and communication between customers, drivers, and management.

### Why was it built:
This project was built to gain practical experience in designing and implementing a **3-tier system**.  
The goal was to learn web technologies using **React** for the frontend and **ASP.NET Core Web APIs** for the backend, while integrating a relational database.

This was built as a **university group project** to learn collaboration, real-world system design, and separation of concerns in a scalable system.

## Features List:

1. Moving Job Booking  
   - Customers can request quotes and book moving jobs.

2. Automated Quote Generation  
   - Quotes are generated based on distance, inventory, and service options.

3. Online & EFT Payments  
   - Customers can pay online or upload proof of EFT payment.

4. Real-Time Job Tracking  
   - Live job progress updates and notifications.

5. Driver Mobile Application  
   - Drivers can manage jobs and communicate with customers.

6. Manager Dashboard & Reporting  
   - Job management, analytics, pricing control, and payment approval.

7. Fleet Management  
   - Manage trucks and service intervals.
